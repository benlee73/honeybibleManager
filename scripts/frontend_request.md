# 프론트엔드 생성 요청서

## 배경
- 카카오톡 오픈채팅방 “성경일독” 대화 CSV를 업로드해 분석 결과 CSV를 내려받는 웹 도구가 필요합니다.
- 방 규칙: 방장이 매일 읽을 분량 공지, 멤버는 날짜+개인 이모티콘으로 인증 (예: `2/4🐽`).

## 목표
- 사용자가 CSV를 업로드하고 분석을 요청하면, 서버가 생성한 결과 CSV를 다운로드할 수 있도록 UI를 제공합니다.

## 요구사항
- CSV 업로드 입력칸(파일 선택)
- “분석” 버튼
- 분석 완료 시 결과 CSV 다운로드 버튼 활성화
- 분석 진행 중 로딩/비활성 상태 표시
- 업로드 파일 확장자 검증(`.csv`)
- 오류 발생 시 사용자에게 메시지 표시

## 사용자 흐름
1) CSV 파일 선택
2) “분석” 버튼 클릭
3) 서버 분석 완료
4) 결과 CSV 다운로드 버튼 활성화 → 다운로드

## 화면 구성(최소)
- 제목/설명
- 파일 업로드 영역
- 분석 버튼
- 상태 메시지 영역(진행/완료/오류)
- 결과 다운로드 버튼

## 입출력
- 입력: 카카오톡 대화 CSV 파일
- 출력: 서버에서 제공하는 결과 CSV 다운로드 링크

## 완료 기준
- CSV 업로드 후 분석 요청 가능
- 분석 완료 시 다운로드 버튼이 활성화되고 결과 CSV를 정상 다운로드
- 오류/진행 상태가 명확히 표시됨

## 참고
- 예시 파일명: `KakaoTalk_Chat_🍯 2026 성경일독 part1_2026-02-04-18-18-23.csv`
