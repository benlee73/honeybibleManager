# 백엔드 생성 요청서

## 배경
- 카카오톡 오픈채팅방 “성경일독” 대화 CSV를 분석해 멤버별 인증 날짜를 추출하고 CSV로 반환해야 합니다.
- 방 규칙: 멤버는 지정 분량을 읽고 날짜+개인 이모티콘으로 인증 (예: `2/4🐽`).

## 목표
- 업로드된 대화 CSV를 분석해, 멤버별로 “이모티콘 + 인증 날짜 목록”을 추출한 결과 CSV를 생성/반환합니다.

## 요구사항
- CSV 업로드 처리(파일 수신)
- 대화 로그에서 멤버별 인증 메시지 추출
- 멤버별 “지정 이모티콘” 식별
- 멤버별 인증 날짜 목록 정리
- 결과 CSV 생성 및 반환
- 입력 파일이 잘못되었을 때 오류 메시지 제공

## 분석 규칙(기본)
- 인증 메시지 패턴: `M/D` 또는 `MM/DD` + 개인 이모티콘
  - 예: `2/4🐽`, `02/04🐽`
- 멤버별 이모티콘은 해당 멤버 메시지에 포함된 이모티콘을 기준으로 식별
- 날짜 목록은 중복 제거 후 정렬(정렬 기준 협의 가능)

## 출력 CSV 포맷(예시)
- 컬럼: `이름 | 이모티콘 | 인증날짜목록`
- 예시 행:
  - `광천 유영훈 | 🐽 | 2/4 | 2/5 | 2/6`
  - `광천 임수아 | 🐰 | 2/4 | 2/7`

## API 개요(예시)
- `POST /analyze` (multipart/form-data)
  - 요청: CSV 파일
  - 응답: 결과 CSV 파일 다운로드

## 완료 기준
- CSV 업로드 시 분석이 수행되고 결과 CSV가 반환됨
- 멤버별로 이모티콘과 날짜 목록이 정확히 정리됨
- 입력 오류 시 명확한 에러 응답 제공

## 참고
